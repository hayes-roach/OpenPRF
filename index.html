[ OPENPRF ]
1Export your save from the console and decrypt the outer PS4 container at garlicsaves.com/decrypt.
2Drop the decrypted savegame folder on the left — any game works. The title and icon are read from sce_sys/param.sfo and icon0.png.
3Save & Download exports one file at a time. If you loaded the whole save folder, use Download all (.zip) to grab everything at once. If you only opened a single file, download it with Save & Download and manually zip it back together with the rest of your save. Re-encrypt at garlicsaves.com/encrypt before copying back to the console.
